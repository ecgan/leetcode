# Power of Two

LeetCode #: [231](https://leetcode.com/problems/power-of-two/)

Difficulty: Easy

Topics: Math, Bit Manipulation.

## Problem

Given an integer, write a function to determine if it is a power of two.

Example 1:

```text
Input: 1
Output: true 
Explanation: 2^0 = 1
```

Example 2:

```text
Input: 16
Output: true
Explanation: 2^4 = 16
```

Example 3:

```text
Input: 218
Output: false
```

## Complexity Analysis

Time complexity: O(1)

Space complexity: O(1)
